## 1. Platform Overview

Notion is an all-in-one AI workspace that combines note-taking, documentation, databases, project management, wikis, and knowledge management into a single platform. With the launch of Notion 3.0 (September 2025), the platform evolved from a productivity tool into an AI-powered workspace with autonomous agents.

### Core Components

| Component | Description |
|---|---|
| Pages | Flexible documents built from modular content blocks |
| Databases | Relational databases with 8 view types, properties, formulas, and automations |
| Wikis | Verified knowledge bases with ownership and verification workflows |
| Teamspaces | Organized areas within a workspace for teams and projects |
| AI | Built-in AI for writing, search, Q&A, autofill, and autonomous agents |
| Sites | Publish Notion pages as public websites with custom domains |
| Calendar | Google Calendar integration for time and task management |
| Mail | Gmail integration with AI-organized inbox |
| Forms | Capture input and feedback directly into databases |

### Deployment

| Aspect | Details |
|---|---|
| Deployment model | Cloud (SaaS) only |
| Access methods | Web app, desktop apps (Mac/Windows), mobile apps (iOS/Android) |
| Content unit | Pages (composed of blocks) and database entries |
| AI models | Multiple premium models including GPT-5, Claude Opus 4.1, o3; user-selectable per task |
